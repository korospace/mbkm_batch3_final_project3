## How to run this project at your local server

1. Setup your database in /config/config.json at development section
2. Run command line below
    ```bash
    npm install
    npm run db:create
    npm run db:migrate
    npm run db:seed
    npm run start
    ```
3. The project will start at http://localhost:4000
